# Covid-19-Italia
#### Consegna per DEXMA
Informazioni riguardanti la situazione covid-19 in Italia, statistiche degli ultimi dieci giorni esposte in grafici e tabelle prelevate da file csv.

### Qui troverai
```
*Dati sui contagi
*Dati sui deceduti
*Grafici 
*Tabelle
*rapporto giornaliero nuovi casi/ tamponi effettuati
```
